## FAQ קצר 💬

**אתה עושה גם “עבודה קטנה”?**  
כן — אם זה משהו שאפשר לבצע מקצועית ונכון, עבודה קטנה יכולה להיות בדיוק מה שמציל את היום.

**אתה נותן הצעת מחיר בלי להגיע?**  
אם יש תמונות/וידאו טובים והרבה פרטים — בהרבה מקרים כן. אם לא, מגיעים לאבחון קצר.

**מה לגבי חשמל?**  
התקנות בסיסיות כן (כמו גופי תאורה/החלפות פשוטות). דברים מורכבים/לוחות/תשתיות — לפי מה שבטוח ונכון לבצע.
